# less-tool

## introduction

	less-tool 是一个把less编译成css工具，它支持自动查找依赖，并自动压缩，支持sourcemap查询
	同时支持 sass编译
## How to install

```
	npm install gulp -g
	
	git clone https://github.com/foreverForCode/less-tool.git 

	cd less-tool

	npm install

	gulp
```

## How to use 

```
gulp  支持 自动监听less

gulp auto-sass 支持 自动监听sass

```
